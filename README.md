# Nimiq Telegram bot
A telegram bot build with the Telegraf package in TypeScript. The bot can be found in the [official Nimiq Channel](https://t.me/nimiq).

# Build and run locally
`npm install`

`npx tsc`

`node dist/app.js`

# Build and watch locally
`npm install`

`npx tsc --watch`

`nodemon dist/app.js`
